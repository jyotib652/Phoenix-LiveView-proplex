Some useful commands
--------------------------------------------

1. To show all the routes:
$ mix phx.routes 

2. Start Phoenix server/application:
$ mix phx.server

3. Using generator to use Auth related code:
$ mix phx.gen.auth Accounts User users 

[Accounts - the new context we're creating; Users - name of the schema for our logged in users;
    users - name of the corresponding database table]

After this command, it will prompt:
An authentication system can be created in two different ways:
- Using Phoenix.LiveView (default)
- Using Phoenix.Controller only
Do you want to create a LiveView based authentication system? [Yn]

Go for LiveView (default) based authentication systems: Y 

After the generator is done with creating authentication system, we need to:
Please re-fetch your dependencies with the following command:

    $ mix deps.get

Remember to update your repository by running migrations:

    $ mix ecto.migrate

Once you are ready, visit "/users/register"
to create your account and then access "/dev/mailbox" to
see the account confirmation email.
